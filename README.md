# QgdUrsApp - QMaps Audit Ursus

Aplicație audit pentru Divizia Ursus - Quatro Grup Distribuție.

## Funcționalități
- Harta clienți cu audit DN (Distribuție Numerică)
- Prezența produse și conformitate
- Scadențar cu grupare pe client și limite credit
- Census clienți activi
- Module escaladări și contracte
- Targeturi și rapoarte vânzări

## Deploy
- **Platform**: Render (auto-deploy on push)
- **Database**: SQLite (better-sqlite3)
